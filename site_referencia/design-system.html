CRT Terminal

Design system for Bruno Reinaldo's portfolio — a Pip-Boy / analog CRT aesthetic powered by monospace typography, green phosphor glow, and zero-pixel-radius precision. This is the single source of truth for colors, components, and patterns.

01 // foundation

Design principles

Five rules that govern every component decision in this system.
p1

Monochrome,
green only

Black background + green phosphor foreground. No blue, purple, red, or warm tones. Status uses green hue variants only.
p2

Analog feel,
digital precision

CRT scanlines, vignette, flicker — but 2px crisp borders, no rounding, no blur. The tube is analog; the grid is exact.
p3

One typeface,
two purposes

JetBrains Mono for everything. Weight + letter-spacing + case differentiate hierarchy, not font changes.
p4

Glow on
interaction

Elements are flat by default. Phosphor glow activates on hover, focus, and active states — energy follows intent.
p5

Boot before
content

Every page starts with a terminal boot sequence. The interface reveals itself with deliberate delay — the screen warms up like hardware.

07 // system

CRT overlay

A fixed overlay (.crt) covers the entire viewport with pointer-events: none. It combines scanlines + vignette. A second pseudo-element (body::before) adds a subtle flicker animation. These are present on every page.
layer stack
z-index 10000 — Boot screen (fades after 2.8s) z-index 9999 — .crt (scanlines + vignette, pointer-events: none) z-index 9998 — body::before (flicker animation, .12s interval) z-index 100 — nav (sticky, glassmorphism) z-index 1 — content sections z-index 0 — .grid-bg (background grid pattern)
scanlines

Horizontal lines

repeating-linear-gradient every 2px with 2.5% opacity green. Simulates CRT phosphor raster.
vignette

Edge darkening

radial-gradient(ellipse at center, transparent 55%, rgba(0,0,0,.8) 100%). Plus ::after for curvature simulation.
flicker

120ms cintilation

body::before with opacity between .97 and 1 at .12s intervals. Nearly imperceptible — feels like a real tube.
.crt{ position:fixed;inset:0;z-index:9999;pointer-events:none; background: repeating-linear-gradient(0deg,transparent 0px,transparent 2px,rgba(0,255,0,.025) 2px,rgba(0,255,0,.025) 4px), radial-gradient(ellipse at center,transparent 55%,rgba(0,0,0,.8) 100%); } .crt::after{ content:'';position:fixed;inset:0;pointer-events:none; background:radial-gradient(ellipse at center,transparent 50%,rgba(0,0,0,.4) 80%,rgba(0,0,0,.6) 100%); } body::before{ content:'';position:fixed;inset:0;z-index:9998;pointer-events:none; animation:crt-flicker .12s infinite;background:transparent; } @keyframes crt-flicker{ 0%{opacity:.98}2%{opacity:1}4%{opacity:.97}6%{opacity:1}10%{opacity:.99} 20%{opacity:1}30%{opacity:.98}40%{opacity:1}50%{opacity:.99}60%{opacity:1} 70%{opacity:.98}80%{opacity:1}90%{opacity:.99}100%{opacity:.98} }
